# Portfolio Windows - Pedro Lucas de Martino

Portfólio pessoal em React inspirado em uma experiência de desktop Windows, criado para apresentar Pedro Lucas de Martino como Desenvolvedor Frontend em formação.

## Sobre

Pedro Lucas de Martino é Desenvolvedor Frontend com foco em ReactJS, TypeScript, JavaScript, Next.js e TailwindCSS. O portfólio reúne bio, currículo, competências, contatos e projeto publicado no GitHub em uma interface com janelas, menu iniciar, taskbar, terminal e explorador de arquivos.

## Conteúdo

- Bio profissional baseada no LinkedIn e currículo
- Currículo real disponível em `public/CV_PedroLucas.pdf`
- Contatos reais: email, LinkedIn, GitHub e WhatsApp
- Projeto real: `Lyguinees/dnd-character-generator`
- Competências técnicas extraídas do currículo
- Experiência atual na Aptiv e formação em ADS no Centro Universitário Una

## Stack

- React
- Vite
- TailwindCSS
- Framer Motion
- React Icons

## Recursos

- Boot screen com login
- Desktop com ícones e wallpaper
- Janelas com abrir, fechar, minimizar, maximizar e arrastar
- Taskbar com busca funcional e preview de janelas abertas
- Start menu com power menu
- Configurações com toggle de tema para barra de título das janelas
- Terminal interativo com comandos do portfólio
- Download do currículo em PDF
- Layout responsivo

## Rodando localmente

```bash
npm install
npm run dev
```

Depois abra o endereço mostrado no terminal.

> Observação: a versão atual do Vite usada pelo projeto exige Node.js 20.19+ ou 22.12+.

## Build

```bash
npm run build
```
