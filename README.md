# Windows 11 Portfolio Prototype

Protótipo de portfólio pessoal em React + TailwindCSS + Framer Motion, inspirado no Windows 11.

## Rodando localmente

```bash
npm install
npm run dev
```

Depois abra o endereço mostrado no terminal.

## Recursos

- Desktop com wallpaper em gradiente
- Ícones estilo Windows
- Janelas com abrir, fechar, minimizar e maximizar
- Drag pela barra superior com Framer Motion
- Foco por z-index dinâmico
- Taskbar glassmorphism com apps ativos
- Start menu simples
- Relógio/data em tempo real
- Boot screen inicial
- Layout adaptado para mobile
- Toggle de som visual, com base pronta para ligar áudio real
